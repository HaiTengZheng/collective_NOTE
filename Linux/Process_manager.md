# 进程 (process)
就是处于执行期的程序。实际上，经常就是正在执行的程序代码的实时结果。
进程是处于执行期的程序以及相关资源的总称。
另一个名字 : 任务 (task)
## 虚拟机制
进程提供两种虚拟机制
- 虚拟处理器
- 虚拟内存

# 执行线程 (thread)
简称线程，是在进程中活动的对象。对于 Linux 而言，线程是一种特殊的进程。
每个线程都拥有一个独立的程序计数器、进程栈和一组进程寄存器。
内核调度的对象是线程。

# fork()
实际由 clone() 系统调用实现

# exec()
创建新的地址空间

# 进程列表
存放在任务队列 (task list) 的双向循环链表中
链表中的每一项都是类型为 task_struct、称为进程描述符的结构
> <linux/sched.h>
